# 股票分析系统 — 用户使用指南

## 快速上手

### 首次登录

1. 在浏览器中打开应用
2. 输入用户名和密码（由管理员提供）
3. 输入 AI API Key（至少填一个）：
   - **Claude API Key**（格式：`sk-ant-api03-...`）— 可在 [Anthropic Console](https://console.anthropic.com/) 获取
   - **MiniMax API Key**（可选）— 用于切换到 MiniMax M2.5 模型
   - **Finnhub API Key**（可选）— 用于美股新闻获取，留空则使用服务器默认配置。可在 [Finnhub](https://finnhub.io/) 免费注册获取
   - 没有任何 AI API Key，所有 AI 功能将无法使用

> API Key 会持久保存在你的个人数据库中，只需输入一次。再次登录时输入新 Key 会覆盖旧值；不输入则保留之前的。

## 新用户推荐流程

```
第 1 步：添加股票        → /stocks 页面，建立你的自选股池
第 2 步：收集数据        → 刷新行情 或 手动上传财务数据
第 3 步：AI 补全数据     → 批量或逐只补全缺失的财务指标
第 4 步：设定投资原则     → /principles 页面，定义你的投资纪律
第 5 步：设置总资金      → /trades 页面，点击总资金卡片输入你的投资总额
第 6 步：日常使用        → 首页洞察、新闻分析、AI 对话
第 7 步：纪律化交易      → /trades 页面，交易前让 AI 审查
第 8 步：AI 模拟操盘     → 首页执行 AI 交易，对比你和 AI 的决策
```

## 页面与功能详解

### 1. 首页仪表盘（`/`）

首页展示投资组合概览：
- 股票数量、财务记录数、投资原则数、对话历史数
- 行业分布图、市值分布图
- 最近添加的股票列表
- 用户持仓价值 vs AI 持仓价值对比

**每日交易建议**
- 基于 5 维度评分体系，展示所有股票的评分和操作建议
- 持仓股票优先排列，显示当前持仓数量和盈亏
- 持仓评分分布图（可视化持仓质量）
- 点击"AI 推理"查看 AI 对评分的详细解读
- 点击"执行 AI 交易"触发 AI 模拟账户的当日交易

**AI 功能：每日持仓洞察**
- 点击生成今日 AI 投资建议
- 识别 2-3 只最具吸引力的持仓（附理由）+ 1-2 只需关注的风险持仓
- 当天缓存，刷新页面不会重复调用 AI

### 2. 股票池（`/stocks`）

你的自选股列表。支持美股（`AAPL`）、A 股（`SH600519`）、港股（`HK09888`）。

**添加股票**：
- 点击"添加股票"按钮，输入股票代码
- 支持搜索功能：输入中文名、拼音首字母或代码均可搜索（如输入"茅台"或"mt"）
- 支持批量添加多只股票

**添加股票后**，系统自动获取：
- 当前价格、市值、PE、PB、股息率、所属行业
- 数据源优先级：雪球 → Yahoo Finance → SEC EDGAR

**工具栏按钮**：

| 按钮 | 功能 |
|------|------|
| 刷新行情 | 批量刷新所有股票的最新价格和市值 |
| **AI 批量补全** | 批量补全多只股票的缺失财务数据 |
| 手动上传 | 从 Excel 表格录入财务数据 |
| 导出 JSON | 导出整个股票池（含财务数据）为 JSON 文件 |
| 导入 JSON | 从 JSON 文件导入股票池（替换当前数据） |

**6 大基本面分析维度**（点击股票行展开查看）：
1. **基础门槛** — 市值、PE、上市年限、持续盈利能力
2. **盈利质量** — 经营现金流、净利润、利润率（3 年趋势）
3. **财务健康** — 现金、负债、总资产、股东权益
4. **业务构成** — 按业务线拆分的营收/利润率
5. **护城河/竞争优势** — 毛利率趋势、市占率
6. **资本配置** — 分红、回购、并购、新业务投资

> 展开后的财务数据支持**点击编辑** — 直接修改单个字段值。

#### AI 补全（单只）

每只股票操作列有魔法棒按钮，点击后：
1. 自动抓取 macrotrends.net、stockanalysis.com 的财务数据
2. Claude AI 从抓取的文本中提取缺失字段
3. 弹出审核界面，展示补全结果

#### AI 批量补全

点击工具栏"AI 批量补全"按钮：
1. 弹出选择界面，列出所有股票及补全状态（已补全/未补全）
2. 支持快捷操作：**全选 / 全不选 / 只选未补全**
3. 确认后逐只处理，实时显示进度和日志
4. **无冲突字段**自动保存，**有冲突字段**汇总后进入人工审核
5. 冲突审核界面默认选择 AI 值，可逐项切换为保留原值
6. 遇到 AI 服务繁忙会自动等待重试（最多 3 次）
7. 支持中途取消

**关于冲突**：当 AI 提取的值与已有数据不同（差异 > 1%）时标记为冲突。常见原因：
- 数据口径差异（如应付账款的狭义 vs 广义定义）
- 数据源本身不一致（雪球对部分美股的费用拆分不准确）
- 非标财年（如 QCOM 财年截止 9 月，雪球可能误取季度数据）

> ETF/基金（如 QQQ、GLD）会自动跳过，因为没有财务报表。

### 3. 媒体中心（`/media`）

汇总股票新闻并运行 AI 情绪分析。

**新闻来源**：
- 美股：Finnhub API（近 7 天公司新闻）
- A 股 / 港股：东方财富 API

**AI 功能：新闻分析**

点击"分析新闻"，为股票池中所有股票触发 AI 分析（5 只并发）：

每只股票的分析输出：
- **情绪判断**：看多 / 看空 / 中性
- **综合摘要**：300 字以内的基本面影响分析
- **关键事件**：3-5 条最重要的事件
- **原则影响**：新闻与你的投资原则的关联

> 新闻分析是**每日快照** — 过期分析会在午夜自动清除。当天已分析过的股票不会重复分析。

### 4. AI 投资助手（`/agent`）

与 Claude 进行多轮对话，讨论你的投资组合。

**创建对话时可选择**：
- **上下文模式**："全局"（所有股票）或"个股"（单只股票）
- **包含投资原则**：开启后会将你的投资纪律注入对话

**对话中的功能**：
- 用 `@股票代码` 提及股票（如"对比一下 @AAPL 和 @NVDA"）
- 实时流式响应
- 从 AI 回复中提炼投资原则

**重要设计理念**：AI 助手 **不能联网**。所有数据均来自你的本地数据库：
- **数据可控**：AI 只分析你亲自验证和收集的数据
- **杜绝幻觉**：股价和财务数据来自你的数据库，不会凭空编造
- **可复现**：相同数据 = 相同分析上下文

> 想让 AI 获得最新数据？请先刷新股票行情或运行新闻分析，然后再开始对话。

### 5. 交易日志（`/trades`）

记录买卖操作，获取 AI 合规审查，跟踪 AI 模拟操盘表现。

页面分为三个区域：**用户交易统计**、**AI 模拟账户统计**、**交易记录/AI 交易历史**。

#### 设置总资金

点击页面顶部"总资金"卡片，输入你的投资总额（美元）。总资金用于：
- 计算用户可用现金（总资金 - 持仓价值）
- AI 模拟账户的资金基准（决定 AI 能买多少）

#### 手动交易记录

- 选择股票、操作方向（买入/卖出）、价格、数量、日期
- 填写交易理由（必填）
- 提交后 AI 自动分析交易是否符合你的投资纪律

AI 返回：
- **违规项**：此交易违反了哪些原则（如有）
- **风险评分**：0-100（越高越危险）
- **分析摘要**：2-3 句话的总结
- **改进建议**：如何优化这次交易决策

> 把它当作"交易前检查清单" — 在实际下单前，让 AI 挑战你的理由。

#### 持仓汇总

- 根据所有买卖记录自动计算当前持仓
- 显示每只股票的持有数量、平均成本、当前盈亏

#### CSV 导入

支持从 Robinhood 导出的 CSV 文件批量导入交易记录（自动识别买卖方向、价格、数量）。

#### AI 模拟操盘

系统维护一个独立的 AI 模拟投资账户，与你的真实持仓分开，让 AI 像基金经理一样自主操盘：

**账户统计卡片**：
- AI 总资产、AI 持仓价值、AI 可用现金
- 与用户持仓价值并列展示，方便对比

**AI 交易决策流程**：
1. 在首页仪表盘点击"执行 AI 交易"
2. AI 综合分析每只股票的评分、近期新闻、你的投资原则
3. 考虑 AI 当前持仓和可用现金，给出具体买卖操作
4. 每笔操作附带 30-50 字的决策理由（引用具体新闻或数据）
5. 每天只执行一次，避免频繁交易

**AI 交易历史**：
- 在交易日志页面底部查看 AI 的每笔操作
- 显示日期、股票、方向（买入/卖出）、数量、价格、决策理由

**重置 AI 账户**：
- 点击"重置AI账户"按钮
- 清空所有 AI 交易记录
- 用你当前的持仓重新初始化 AI 账户（与你起点一致）
- 适合想让 AI 从你当前状态重新开始的场景

**5 维度评分体系**（驱动 AI 交易决策，用户可自定义权重）：

| 维度 | 默认权重 | 评估内容 |
|------|---------|---------|
| 估值 | 30% | PE、PB、PS、股息率 |
| 盈利质量 | 25% | 连续盈利、利润率、现金流/利润比 |
| 财务健康 | 20% | 净现金、负债率、流动性 |
| 护城河 | 15% | 毛利率趋势、市场份额 |
| 新闻情绪 | 10% | 当日新闻分析的看多/看空/中性 |

评分对应操作建议：≥80 买入/加仓，60-80 持有，<40 减仓/卖出。

> AI 模拟操盘的目的是让你对比自己的交易决策和 AI 的建议，长期跟踪谁的收益更好。AI 不会帮你执行真实交易，所有操作仅在模拟账户中进行。

### 6. 我的投资原则（`/principles`）

你的个人投资纪律手册，贯穿所有 AI 功能。

**每条原则包含**：
- 标题（简短，如"不追涨杀跌"）
- 内容（具体规则，如"PE 超过 40 的股票，除非营收增速 > 30%，否则不买入"）
- 分类：风险 / 估值 / 选股 / 行为
- 启用开关（禁用的原则不会注入 AI 上下文）

**原则被用于**：
- AI 对话（开启时注入）
- 交易分析（逐条检查违规）
- 新闻分析（影响评估）
- AI 模拟交易（决策参考）
- 首页每日洞察

**如何建立原则**：
1. 与 AI 讨论你的投资理念
2. 在有价值的回复上点击"提炼投资原则"
3. 审核、编辑后保存
4. 也可以手动添加自己的规则

### 7. 财报管理（`/financial_report`，高级功能）

> 此功能已从导航栏隐藏，可通过直接访问 `/financial_report` 使用。日常使用建议优先用"AI 补全"获取财务数据，更快且消耗更少 AI 额度。

从 SEC EDGAR 下载 10-K/10-Q 原文，并用 AI 从原文提取财务数据。适合需要查看财报原文或对数据准确性有极高要求的场景。

## AI 功能总览

| 功能 | 触发方式 | 数据来源 | 需要联网？ |
|-----|---------|---------|----------|
| 首页每日洞察 | 页面加载 / 点击 | 仅本地数据库 | 否 |
| AI 对话 | 发送消息 | 仅本地数据库 | 否 |
| 提炼投资原则 | 点击按钮 | 本地数据库（最后一条 AI 回复） | 否 |
| 交易分析 | 提交交易 | 仅本地数据库 | 否 |
| AI 模拟交易 | 每日自动 / 手动触发 | 本地数据库 + 评分 + 新闻 | 否 |
| 新闻分析 | 点击"分析" | 外部新闻 API + 本地投资原则 | 是（获取新闻） |
| AI 补全 / 批量补全 | 点击按钮 | 外部网页抓取 + 本地数据库 | 是（网页抓取） |
| 财报分析 | 点击"分析" | 本地文件（已上传/下载） | 否 |
| 股票评分 | 每日自动 | 本地财务数据 + 新闻分析 | 否 |

**核心原则**：大部分 AI 功能仅使用**本地数据**。AI 在对话和交易分析时不会浏览互联网，确保数据完整性，杜绝编造市场数据。

## 自动化定时任务

系统内置两个定时任务（可在 `/api/scheduler/jobs` 查看）：

| 任务 | 时间 | 功能 |
|------|------|------|
| 刷新股价 | 每日凌晨 3:00 | 自动更新所有股票的最新价格和市值 |
| 刷新财务数据 | 每周一凌晨 4:00 | 自动更新所有股票的季度/年度财务数据 |

> 定时任务对所有用户的数据库同时执行。

## 数据来源说明

| 数据源 | 覆盖范围 | 用途 |
|--------|---------|------|
| 雪球 (Xueqiu) | 美股 / A 股 / 港股 | 股票搜索、行情、财务数据（优先） |
| Yahoo Finance | 美股为主 | 行情、财务数据（备选） |
| SEC EDGAR | 美股 | 年报 10-K/10-Q 文件 |
| Finnhub | 美股 | 新闻数据（需要 API Key，登录时可填写或使用服务器默认） |
| 东方财富 | A 股 / 港股 | 新闻数据 |
| macrotrends / stockanalysis | 美股 | AI 补全时抓取财务数据 |

系统采用**智能数据源切换**：优先使用雪球，如果失败则自动尝试下一个数据源。单个数据源连续失败 3 次后进入冷却期。

## 使用技巧

1. **保持数据新鲜**：定期刷新股价和财务数据，或依赖自动定时任务
2. **善用 AI 批量补全**：新建股票池后一键补全所有缺失数据，注意审核冲突
3. **逐步建立原则**：通过对话积累、提炼、完善投资纪律
4. **交易前先审查**：在实际下单前用交易日志让 AI 挑战你的理由
5. **每日查看新闻**：早上运行新闻分析，掌握今日市场情绪
6. **对话中 @ 提及股票**：用 `@SYMBOL` 让 AI 聚焦特定股票
7. **对比 AI 模拟账户**：定期查看 AI 的交易决策和收益，反思自己的策略
8. **仔细审核 AI 补全**：冲突可能源于数据口径差异，不一定是错误
