# 📊 股票筛选功能使用指南

## 🎯 功能概述

股票筛选系统允许你使用预设的投资策略来筛选符合条件的股票。系统已预置5种经典投资策略。

---

## 🚀 快速开始

### 1. 访问筛选页面

启动服务器后，访问：
```
http://localhost:5002/screening
```

### 2. 查看可用策略

页面会显示所有可用的筛选策略，每个策略都包含：
- **策略名称和描述**
- **关键筛选指标**（如市值、ROE、P/E等）
- **启用状态**
- **创建时间**

### 3. 运行筛选

点击策略卡片上的 **"运行筛选"** 按钮，系统会：
1. 从股票池中筛选符合条件的股票
2. 计算每只股票的综合评分
3. 按评分排序显示结果

---

## 📋 预置的5种筛选策略

### 1️⃣ Value Investing (Buffett Style) - 价值投资
**适合**: 寻找稳定盈利、估值合理的优质公司

**筛选条件**:
- 市值 ≥ $100B（大型蓝筹股）
- 上市时间 ≥ 5年
- 连续盈利 ≥ 3年
- 利润率 ≥ 10%
- ROE ≥ 15%
- P/E ≤ 25
- P/B ≤ 3.0
- 流动比率 ≥ 1.5
- 负债率 ≤ 0.5
- 排除金融行业

**特点**: 注重公司质量和财务健康度，适合长期投资

---

### 2️⃣ Growth Investing - 成长投资
**适合**: 寻找高成长性公司

**筛选条件**:
- 市值 ≥ $10B
- 连续盈利 ≥ 2年
- 利润率 ≥ 5%
- 营收增长率 ≥ 15%
- 利润增长率 ≥ 20%
- P/E ≤ 50（可接受较高估值）
- 流动比率 ≥ 1.0
- 负债率 ≤ 1.0
- 排除公用事业和房地产行业

**特点**: 关注高成长，可接受相对较高估值

---

### 3️⃣ Dividend Investing - 股息投资
**适合**: 寻找高股息、稳定分红的公司

**筛选条件**:
- 市值 ≥ $50B
- 上市时间 ≥ 10年
- 连续盈利 ≥ 5年
- 利润率 ≥ 8%
- ROE ≥ 10%
- 股息率 ≥ 3%
- P/E ≤ 20
- 流动比率 ≥ 1.5
- 负债率 ≤ 0.6

**特点**: 注重稳定现金流和分红能力，适合养老金等稳健投资

---

### 4️⃣ Quality at Reasonable Price (GARP) - 合理价格的优质股
**适合**: 在合理价格买入优质成长股

**筛选条件**:
- 市值 ≥ $30B
- 连续盈利 ≥ 3年
- 利润率 ≥ 12%
- ROE ≥ 18%
- 营收增长率 ≥ 10%
- 利润增长率 ≥ 12%
- P/E ≤ 30
- P/B ≤ 4.0
- 流动比率 ≥ 1.2
- 负债率 ≤ 0.7

**特点**: 平衡质量和成长，不追求极端高估值

---

### 5️⃣ Conservative (Safety First) - 保守投资
**适合**: 注重安全性和稳定性的保守投资者

**筛选条件**:
- 市值 ≥ $200B（超大型公司）
- 上市时间 ≥ 15年
- 连续盈利 ≥ 10年
- 利润率 ≥ 15%
- ROE ≥ 12%
- 股息率 ≥ 2%
- P/E ≤ 18
- P/B ≤ 2.5
- 流动比率 ≥ 2.0
- 负债率 ≤ 0.3

**特点**: 极度保守，只选择最稳定的蓝筹股

## 🎨 界面功能详解

### 策略卡片
每个策略卡片显示：
- ⭐ 策略名称（星号表示启用状态）
- 📝 策略描述
- ✅ 关键筛选指标
- 🎯 操作按钮：
  - **运行筛选**: 执行筛选并查看结果
  - **详情**: 查看完整的筛选条件

### 筛选结果表格
显示筛选结果，包括：
- **排名**: 按评分排序
- **代码**: 股票代码
- **名称**: 公司名称
- **行业**: 所属行业
- **市值**: 市场价值
- **P/E**: 市盈率
- **ROE**: 净资产收益率
- **评分**: 综合评分（0-100）
  - 🟢 80-100: 优秀
  - 🟡 60-79: 良好
  - 🔴 0-59: 一般
- **操作**: 查看股票详情

### 评分系统
综合评分基于多个维度：
- 财务健康度（负债率、流动比率）
- 盈利能力（利润率、ROE）
- 成长性（营收增长、利润增长）
- 估值合理性（P/E、P/B）
- 分红能力（股息率）

## 🔧 API使用（程序化访问）

### 获取所有筛选策略
```bash
curl http://localhost:5002/api/screening/criteria
```

**响应示例**:
```json
{
  "success": true,
  "count": 5,
  "data": [
    {
      "name": "Value Investing (Buffett Style)",
      "description": "寻找稳定盈利、估值合理的优质公司",
      "is_active": true,
      "criteria": {
        "market_cap": {"min": 100.0, "max": null},
        "profitability": {
          "min_consecutive_years": 3,
          "min_profit_margin": 10.0,
          "min_roe": 15.0
        },
        "valuation": {
          "max_pe": 25.0,
          "max_pb": 3.0
        }
      }
    }
  ]
}
```

### 运行筛选
```bash
curl -X POST http://localhost:5002/api/screening/run \
  -H "Content-Type: application/json" \
  -d '{"criteria_name": "Value Investing (Buffett Style)"}'
```

**响应示例**:
```json
{
  "success": true,
  "criteria": "Value Investing (Buffett Style)",
  "matches": 3,
  "data": {
    "results": [
      {
        "symbol": "AAPL",
        "name": "Apple Inc.",
        "sector": "Technology",
        "market_cap": 2500.0,
        "pe_ratio": 24.5,
        "roe": 18.2,
        "score": 85
      }
    ]
  }
}
```

## ⚠️ 重要提示

### 数据要求
筛选功能需要股票池中有完整的财务数据。确保：
1. 已添加股票到股票池
2. 已从Yahoo Finance或SEC获取财务数据
3. 数据相对新鲜（建议定期更新）

### 添加股票数据
```bash
# 方法1: 通过Web界面
访问 http://localhost:5002/stocks → 添加股票 → 勾选"自动获取数据"

# 方法2: 通过API
curl -X POST http://localhost:5002/api/stocks \
  -H "Content-Type: application/json" \
  -d '{"symbol": "AAPL", "fetch_data": true}'

# 方法3: 通过CLI
cd backend
source venv/bin/activate
python cli.py add-stock AAPL --fetch
```

### 筛选注意事项
- **空结果**: 如果筛选结果为空，可能是条件太严格或数据不完整
- **评分差异**: 评分是综合指标，不同策略侧重点不同
- **人工审核**: 筛选结果仅供参考，投资前请人工深度分析
- **市场变化**: 定期重新筛选，市场条件会改变

## 💡 小贴士

### 组合使用策略
可以结合多个策略寻找"多重验证"的股票：
1. 先用"Value Investing"筛选一遍
2. 再用"GARP"筛选一遍
3. 找出两次都出现的股票
4. 这些股票同时满足价值和成长标准

### 参数调整建议
如果筛选结果太少：
- 降低ROE要求（15% → 12%）
- 放宽P/E限制（25 → 30）
- 减小最小市值（100B → 50B）

如果筛选结果太多：
- 提高ROE要求（15% → 20%）
- 收紧P/E限制（25 → 20）
- 增加最小市值（100B → 200B）

### 行业轮动
- 科技行业：使用 Growth Investing
- 公用事业：使用 Dividend Investing
- 消费品：使用 Value Investing
- 医疗保健：使用 GARP
