点击"添加股票"按钮，弹出对话框：

```
┌─────────────────────────────────────┐
│  添加股票                     ✕     │
├─────────────────────────────────────┤
│                                     │
│  股票代码 *                         │
│  [____________]  例如: AAPL         │
│  请输入美股代码                     │
│                                     │
│  公司名称（可选）                   │
│  [____________]  例如: Apple Inc.   │
│                                     │
│  ☑ 自动从Yahoo Finance获取数据      │
│                                     │
├─────────────────────────────────────┤
│             [取消]  [添加]          │
└─────────────────────────────────────┘
```

**步骤：**
1. 输入股票代码（必填）
2. 输入公司名称（可选）
3. 选择是否自动获取数据
4. 点击"添加"

## 💡 使用技巧

### 1. 快速添加多只股票
```javascript
// 打开浏览器Console (F12)
const symbols = ['AAPL', 'MSFT', 'GOOGL', 'AMZN'];
symbols.forEach(symbol => {
    api.stocks.add({symbol: symbol})
        .then(() => console.log(`${symbol} added`));
});
```

### 2. 批量刷新数据
点击每只股票的刷新按钮，等待数据更新。

### 3. 查看实时统计
仪表盘会自动显示最新统计数据。

## 🔧 故障排除

### 问题1：页面无法加载
**解决**：
```bash
# 检查服务器是否运行
curl http://localhost:5001/health

# 重启服务器
pkill -f "python run.py"
python run.py
```

### 问题2：图表不显示
**解决**：
1. 检查浏览器Console (F12)
2. 确认Chart.js加载成功
3. 检查是否有数据

### 问题3：添加股票失败
**解决**：
1. 检查股票代码是否正确
2. 查看Network标签的API响应
3. 检查服务器日志
