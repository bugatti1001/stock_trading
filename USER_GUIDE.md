# 股票分析系统 — 用户使用指南

## 快速上手

### 首次登录

1. 在浏览器中打开应用
2. 输入用户名和密码（由管理员提供）
3. 输入你的 **Claude API Key**（格式：`sk-ant-api03-...`）
   - 此 Key 会保存在你的个人数据库中，只需输入一次
   - 可在 [Anthropic Console](https://console.anthropic.com/) 获取
   - 没有此 Key，所有 AI 功能将无法使用

> 登录后 API Key 会持久保存。再次登录时输入新 Key 会覆盖旧值；不输入则保留之前的。

---

## 新用户推荐流程

```
第 1 步：添加股票        → /stocks 页面，建立你的自选股池
第 2 步：收集数据        → 刷新行情 或 手动上传财务数据
第 3 步：设定投资原则     → /principles 页面，定义你的投资纪律
第 4 步：日常使用        → 首页洞察、新闻分析、AI 对话
第 5 步：纪律化交易      → /trades 页面，交易前让 AI 审查
```

## 页面与功能详解

### 1. 首页仪表盘（`/`）

首页展示投资组合概览：
- 股票数量、财务记录数、投资原则数、对话历史数
- 行业分布图、市值分布图
- 最近添加的股票列表

**AI 功能：每日持仓洞察**
- 点击生成今日 AI 投资建议
- 识别 2-3 只最具吸引力的持仓（附理由）+ 1-2 只需关注的风险持仓
- 当天缓存，刷新页面不会重复调用 AI

| 数据来源 | 说明 |
|---------|------|
| 本地数据库 | 所有股票 + 近 3 年财务数据 |
| 本地数据库 | 今日新闻分析结果 |
| 本地数据库 | 你的有效投资原则 |

---

### 2. 股票池（`/stocks`）

你的自选股列表。通过股票代码添加（如 `AAPL`、`SH600519`、`HK09888`）。

**添加股票后**，系统自动获取：
- 当前价格、市值、PE、PB、股息率、所属行业（来自 Yahoo Finance）

**财务数据采集**（可选择一种或多种方式）：
- **刷新数据**：从 Yahoo Finance 拉取季度/年度财务数据
- **手动上传**：通过 `/stocks/manual_upload` 页面从 Excel 表格录入
- **AI 补全**：让 AI 从财经网站抓取并提取缺失的财务数据

**6 大基本面分析维度**（根据财务数据自动计算）：
1. **基础门槛** — 市值、PE、上市年限、持续盈利能力
2. **盈利质量** — 经营现金流、净利润、利润率（3 年趋势）
3. **财务健康** — 现金、负债、总资产、股东权益
4. **业务构成** — 按业务线拆分的营收/利润率
5. **护城河/竞争优势** — 毛利率趋势、市占率
6. **资本配置** — 分红、回购、并购、新业务投资

**AI 功能：AI 补全**

当财务数据有缺失字段时，点击"AI 补全"自动填充：

| 数据来源 | 说明 |
|---------|------|
| 外部网页 | 抓取 macrotrends.net、stockanalysis.com 的财务数据 |
| 外部 API | Google Custom Search（按需使用） |
| 本地数据库 | 已有的部分数据（用于冲突检测） |
| Claude AI | 从抓取的文本中提取结构化数字 |

AI 会展示结果供你审核：
- **自动填充**：原本为空的字段，由 AI 填入（自动接受）
- **冲突项**：已有值与 AI 值不同（由你选择保留哪个）
- **未找到**：AI 无法提取的字段（保持为空）

---

### 3. 媒体中心（`/media`）

汇总股票新闻并运行 AI 情绪分析。

**新闻来源**：
- 美股：Finnhub API（近 7 天公司新闻）
- A 股 / 港股：东方财富 API

**AI 功能：新闻分析**

点击"分析新闻"，为股票池中所有股票触发 AI 分析：

| 数据来源 | 说明 |
|---------|------|
| 外部 API | Finnhub / 东方财富的新闻标题和摘要 |
| 本地数据库 | 你的有效投资原则 |
| Claude AI | 分析每只股票的情绪和影响 |

每只股票的分析输出：
- **情绪判断**：看多 / 看空 / 中性
- **综合摘要**：300 字以内的基本面影响分析
- **关键事件**：3-5 条最重要的事件
- **原则影响**：新闻与你的投资原则的关联

> 新闻分析是**每日快照** — 过期分析会在午夜自动清除。
> 当天已分析过的股票不会重复分析。

### 4. AI 投资助手（`/agent`）

与 Claude 进行多轮对话，讨论你的投资组合。这是核心 AI 功能。

**创建对话时可选择**：
- **上下文模式**："全局"（所有股票）或"个股"（单只股票）
- **包含投资原则**：开启后会将你的投资纪律注入对话

**对话中的功能**：
- 用 `@股票代码` 提及股票（如"对比一下 @AAPL 和 @NVDA"）
- 实时流式响应
- 从 AI 回复中提炼投资原则

| 数据来源 | 说明 |
|---------|------|
| 本地数据库 | 股票基本面（近 3 年财务、6 大分析维度） |
| 本地数据库 | 今日新闻分析结果 |
| 本地数据库 | 你的投资原则（如已开启） |
| 本地数据库 | 对话历史（最近 20 条消息） |

**重要设计理念**：AI 助手 **不能联网**。所有数据均来自你的本地数据库。这是有意为之的设计：

- **数据可控**：AI 只分析你亲自验证和收集的数据
- **杜绝幻觉**：股价和财务数据来自你的数据库，不会凭空编造
- **可复现**：相同数据 = 相同分析上下文
- **隐私安全**：你的持仓数据不会外泄

> 想让 AI 获得最新数据？请先刷新股票行情或运行新闻分析，然后再开始对话。

**AI 功能：提炼投资原则**

收到有价值的 AI 回复后，点击"提炼投资原则"：
- AI 阅读最后一条助手消息
- 提取可操作的、第一人称的原则（如"我不会在 PE > 40 时追高"）
- 由你审核确认后再保存

### 5. 交易日志（`/trades`）

记录买卖操作，获取 AI 合规审查。

**记录交易**：
- 选择股票、操作方向（买入/卖出）、价格、数量、日期
- 填写交易理由（必填）

**AI 功能：交易分析**

| 数据来源 | 说明 |
|---------|------|
| 本地数据库 | 完整股票池的基本面数据 |
| 本地数据库 | 你的所有有效投资原则 |
| 本地数据库 | 今日新闻分析 |
| 本地数据库 | 该股票的最新行情 + 财务数据 |
| Claude AI | 评估交易是否符合你的投资纪律 |

AI 返回：
- **违规项**：此交易违反了哪些原则（如有）
- **风险评分**：0-100（越高越危险）
- **分析摘要**：2-3 句话的总结
- **改进建议**：如何优化这次交易决策

> 把它当作"交易前检查清单" — 在实际下单前，让 AI 挑战你的理由。

---

### 6. 我的投资原则（`/principles`）

你的个人投资纪律手册，贯穿所有 AI 功能。

**每条原则包含**：
- 标题（简短，如"不追涨杀跌"）
- 内容（具体规则，如"PE 超过 40 的股票，除非营收增速 > 30%，否则不买入"）
- 分类：风险 / 估值 / 选股 / 行为
- 启用开关（禁用的原则不会注入 AI 上下文）

**原则被用于**：
- AI 对话（开启时注入）
- 交易分析（逐条检查）
- 新闻分析（影响评估）
- 首页每日洞察

**如何建立原则**：
1. 与 AI 讨论你的投资理念
2. 在有价值的回复上点击"提炼投资原则"
3. 审核、编辑后保存
4. 也可以手动添加自己的规则

---

### 7. 财报管理（`/financial_report`）

管理年报/季报文件（10-K、10-Q、20-F）。

**操作**：
- **获取**：从 SEC EDGAR 查询最新年报元数据（仅限美股）
- **下载**：从 SEC 拉取实际报告文件
- **上传**：手动上传 PDF/HTML（适用于任何股票）
- **分析**：AI 从报告文件中提取财务数据

## AI 功能总览

| 功能 | 触发方式 | 数据来源 | 需要联网？ |
|-----|---------|---------|----------|
| 首页每日洞察 | 页面加载 / 点击 | 仅本地数据库 | 否 |
| AI 对话 | 发送消息 | 仅本地数据库 | 否 |
| 提炼投资原则 | 点击按钮 | 本地数据库（最后一条 AI 回复） | 否 |
| 交易分析 | 提交交易 | 仅本地数据库 | 否 |
| 新闻分析 | 点击"分析" | 外部新闻 API + 本地投资原则 | 是（获取新闻） |
| AI 补全 | 点击"AI 补全" | 外部网页抓取 + 本地数据库 | 是（网页抓取） |
| 财报分析 | 点击"分析" | 本地文件（已上传/下载） | 否 |

**核心原则**：大部分 AI 功能仅使用**本地数据**。AI 在对话和交易分析时不会浏览互联网，确保数据完整性，杜绝编造市场数据。

仅有两个例外（新闻分析和 AI 补全）会先获取外部数据，再交给 AI 进行结构化提取。保存前你始终可以审核确认。

## 使用技巧

1. **保持数据新鲜**：定期刷新股价和财务数据
2. **逐步建立原则**：通过对话积累、提炼、完善投资纪律
3. **交易前先审查**：在实际下单前用交易日志让 AI 挑战你的理由
4. **每日查看新闻**：早上运行新闻分析，掌握今日市场情绪
5. **对话中 @ 提及股票**：用 `@SYMBOL` 让 AI 聚焦特定股票
6. **仔细审核 AI 补全**：确认冲突解决界面后再保存
